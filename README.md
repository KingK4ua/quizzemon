⚡ Quizzemon 
O Quizzemon é uma plataforma de quizzes interativos inspirada na interface de aprendizado gamificada do Duolingo, mas com o foco total no universo Pokémon. O projeto foi desenvolvido para testar os conhecimentos dos treinadores através de uma jornada visualmente rica e intuitiva.

🚀 Funcionalidades
Interface Gamificada: Trilha de progresso em formato de onda, simulando o caminho de um treinador Pokémon.

Sistema de Unidades: Conteúdo dividido por seções e unidades para organizar os desafios.

Feedback em Tempo Real: Barra de progresso dinâmica que aumenta conforme o acerto das questões.

Design Responsivo e Moderno: Estética clean ("Tech/Geek") com uso de modais e animações suaves.

Base de Dados Dinâmica: Sistema preparado para carregar diferentes módulos de perguntas.

🛠️ Tecnologias Utilizadas
O projeto foi construído utilizando tecnologias fundamentais de desenvolvimento web:

HTML5: Estruturação semântica da aplicação.

CSS3: Estilização avançada, incluindo o layout de trilha curva (nth-child) e efeitos de profundidade nas Pokébolas.

JavaScript (ES6): Lógica do quiz, manipulação de DOM para progresso e controle dos modais de interação.

🎮 Como Jogar
Acesse a trilha principal.

Clique na Pokébola ativa para abrir o desafio.

Responda as perguntas corretamente para avançar na barra de progresso.

Conclua o quiz para ganhar XP e liberar novos desafios.

📂 Estrutura do Código
O arquivo principal index.html contém toda a estrutura da Sidebar, Área Principal de trilha e as telas de Quiz, garantindo uma navegação de Single Page Application (SPA) fluida.

Nota: Este projeto faz parte de um esforço conjunto para aplicar conceitos de lógica de programação e design de interface em um tema apaixonante para a comunidade geek.
